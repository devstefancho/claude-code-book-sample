# Chapter 01. 리팩터링 워크플로를 구성하는 방법

> 프롬프트: [prompts.md](./prompts.md) | 코드: [code.md](./code.md)

클로드 코드의 고도화된 리팩터링 워크플로(Workflow)를 이해하고 구성하는 방법을 알아봅니다.

## 워크플로 이해하기

효율적인 리팩터링 시스템을 구축하기 위해서는 전체적인 작업 흐름을 이해해야 합니다:
- 규칙(Rules)
- 명령어(Commands)
- 서브 에이전트(Subagent)

## 워크플로의 핵심 요소

1. **일관된 품질 유지**: 규칙과 스킬을 통한 코드 품질 관리
2. **단계적 실행**: 분석 → 계획 → 구현의 단계별 진행
3. **안전장치**: 체크포인트, 워크트리, Plan 에이전트 활용
