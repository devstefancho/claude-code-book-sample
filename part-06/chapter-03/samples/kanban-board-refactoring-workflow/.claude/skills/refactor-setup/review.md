# Code Review

/analyze 결과를 기반으로 상세 코드 리뷰를 진행합니다.

## Review Checklist

### 파일별 분석

- [ ] 현재 코드 구조 파악
- [ ] 문제점 식별 (분석 결과 기반)
- [ ] 의존성 관계 확인
- [ ] 영향 범위 파악

### 리팩토링 방안

- [ ] 구체적인 수정 방법 제시
- [ ] 단계별 실행 순서 정의
- [ ] 테스트 필요 사항 명시

### 주의사항

- [ ] Breaking change 여부
- [ ] 롤백 가능성
- [ ] 사이드 이펙트 예상

## Output Format

리뷰 결과는 다음 형식으로 정리:

```markdown
### [파일명]

**현재 상태:**

- [코드 분석 내용]

**문제점:**

- [구체적 문제]

**권장 수정:**

1. [단계 1]
2. [단계 2]
```
