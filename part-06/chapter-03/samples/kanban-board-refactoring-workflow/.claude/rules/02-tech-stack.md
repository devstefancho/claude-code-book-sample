# 기술 스택

## 프론트엔드

- **Next.js 15**: App Router, Server Components
- **TypeScript**: 정적 타입 검사
- **Tailwind CSS**: 유틸리티 기반 스타일링
- **@dnd-kit**: 드래그 앤 드롭 구현

## 상태관리

- **Zustand**: 전역 상태 관리 (카드, 보드 데이터)

## 백엔드/데이터베이스

- **Supabase**: PostgreSQL 기반 BaaS
- **Supabase Auth**: JWT 기반 인증
- **@supabase/ssr**: 서버 사이드 Supabase 클라이언트

## 검증

- **Zod**: 런타임 스키마 검증

## 테스트

- **Vitest**: 단위 테스트 프레임워크
- **@testing-library/react**: 컴포넌트 테스트

## 개발 도구

- **ESLint**: 코드 품질 검사
- **Prettier**: 코드 포맷팅
- **Husky + lint-staged**: Git 훅으로 자동 검사
