# 프로젝트 개요

## 프로젝트명

칸반 보드 (Kanban Board)

## 목적

팀 또는 개인의 작업을 시각적으로 관리하고 진행 상태를 추적하는 웹 애플리케이션

## 핵심 기능

- 3개 상태 컬럼: 할 일(todo), 진행 중(in-progress), 완료(done)
- 카드 CRUD: 생성, 조회, 수정, 삭제
- 드래그 앤 드롭: 카드를 컬럼 간 이동 및 순서 변경
- 사용자 인증: 회원가입, 로그인, 로그아웃
- 데이터 동기화: Supabase를 통한 실시간 저장
- 낙관적 업데이트: 즉각적인 UI 반응

## 사용자 흐름

1. 회원가입/로그인으로 인증
2. 개인 보드 자동 생성 (최초 접속 시)
3. 카드 생성 및 관리
4. 드래그로 상태 변경
