# Chapter 01. 프롬프트 모음

## 번역 스킬 사용하기

```
[나]: 안녕하세요를 번역 스킬써서 번역해줘

[AI]: "안녕하세요"를 정중한 영어 표현으로 번역한 결과입니다:

  {
      "input": "안녕하세요",
      "result": "Hello" or "Good day" or "Greetings"
  }

  상황별 정중한 표현:
  - 일반적: "Hello"
  - 공식적/비즈니스: "Good morning/afternoon/evening" (시간대에 따라)
  - 격식 있는 인사: "Greetings"
  - 이메일 시작: "Dear [Name]," 또는 "Hello [Name],"

  가장 보편적이고 정중한 표현은 **"Hello"**입니다.
```
