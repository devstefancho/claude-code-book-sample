## Zero Shot 보다는 Few Shot이 효과적이다

- Zero-shot: 예시 없음
- One-shot: 예시 1개
- Few-shot: 예시 여러 개 (Anthropic 권장: 3~5개)
- 코드 리뷰 스킬: 원칙당 2~3개 권장
