## 대화 히스토리 요약하기 예시

```
/compact 현재 코드의 UI 버그에 대해서 논의하던 것에 초점을 맞춰줘
```

## statusline 설정

```
/statusline 표시할 정보:
  - 현재 디렉토리 (~ 축약)
  - 모델명
  - 컨텍스트 사용량 (프로그레스 바로)

  출력 형식: DIR: ~/path | MODEL: Opus 4.5 | CTX: [██░░░░░░░░] 15%
```
