## 병렬 실험을 위한 복합 프롬프트

```
[나]: 지금 프로젝트를 ESM으로 전환하려고 해. 두 가지 실험을 동시에 진행하고 싶어.
먼저 experiment-fast 워크트리를 만들고, 기존 코드를 최소한으로 수정해서 ESM으로만 전환해줘.
그리고 experiment-modular 워크트리를 만들고, ESM 전환과 동시에 기능별로 코드를 더 작게
분리하는 모듈화 리팩터링을 진행해줘. 이렇게 두 작업이 끝나면 각 워크트리의 결과물을 비교해서
어떤 방식이 유지보수 관점에서 더 유리한지 보고해줘.
```
