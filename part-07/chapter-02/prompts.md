# Chapter 02. 프롬프트 모음

## Slack에서 Hermes 소환하기

```
[나] : 안녕

[AI] : 안녕하세요! 무엇을 도와드릴까요?
```

```
[나] : @MyAgent 너는 무슨 모델이야?

[AI] : 저는 OpenAI의 GPT-5.4 기반 어시스턴트예요. 지금 이 Slack 스레드에서 일을 도와드릴 수 있어요.
```

## 크론잡으로 뉴스 브리핑 자동화하기

```
[나] : @MyAgent 크론잡 생성해줘

목표: 최근 AI 업계의 핵심 뉴스 10건을 한국어로 브리핑.

수집 지침:

1. Hacker News, TechCrunch, GeekNews 세 출처를 반드시 확인한다.

2. AI 관련 범주: 모델 출시, 에이전트, LLM, 생성형 AI, 반도체/인프라, 제품 출시, 투자/인수, 정책/규제.

3. 최근 24시간 내 게시물 우선. 부족하면 72시간까지 확장.

4. 중복 주제는 하나로 합치고 가장 정보가 좋은 링크를 대표로 선택.

5. 확인되지 않은 내용은 추측하지 않는다.

6. 총 10개 항목, 중요도 순 정렬.

출력 형식:

오늘의 AI 뉴스 브리핑 (YYYY-MM-DD)

한 줄 브리핑: 오늘 AI 시장/기술 흐름을 1문장으로.

주요 뉴스 10선

1) 짧은 한국어 제목

- 요약: 2문장 이내

- 중요성: 왜 중요한지 1문장

- 출처: 출처명, 원문 링크

오늘의 흐름

- 핵심 트렌드 3~5개

문체: 간결체, 불필요한 수식어 배제.

[AI] : 크론잡 생성이 가능해요. 언제 실행되게 할까요?

[나] : 평일 오전 8시

[AI] : 생성을 완료했습니다.
```

## 크론잡 즉시 실행

```
[나] : @MyAgent 지금 1회 실행해줘

[AI] : 지금 1회 실행 걸어두었습니다.
```

## 크론잡에서 개선포인트 잡아내기

```
[나] : 각 뉴스마다 날짜를 명시해줘, 프롬프트 기준에 부합하지 않은 뉴스는 제외해야한다.

[AI] : 반영했습니다. 원하시면 수정된 기준으로 지금 다시 실행해드릴게요

[나] : 지금 다시 1회 실행해줘

[AI] : 지금 다시 1회 실행했습니다.
```
